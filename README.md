# Nature's Remedy - Medicinal Plants & Benefits


**Description**

Nature's Remedy is an informative and visually appealing website dedicated to exploring the world of medicinal plants and trees. It provides users with detailed information on various plants like Aloe Vera, Neem, Tulsi, Ginger, Turmeric, and Mint, highlighting their benefits, uses, and safe application tips. The site aims to promote natural wellness by bridging traditional herbal knowledge with modern science-backed insights.

This project is built as a static website using HTML, CSS, and JavaScript, making it easy to host and maintain. Whether you're a beginner interested in herbal remedies or someone seeking natural health alternatives, Nature's Remedy serves as a comprehensive resource.

**Features**

Interactive Plant Cards: Browse through a collection of medicinal plants with images, brief descriptions, and expandable "Read More" sections for in-depth information.

Search Functionality: Quickly find specific plants using the search bar that filters cards in real-time.

Dark Mode Toggle: Switch between light and dark themes for a personalized viewing experience.

Responsive Design: Fully responsive layout that works seamlessly on desktops, tablets, and mobile devices, powered by Bootstrap.

Smooth Scrolling Navigation: Navigate between sections (Home, About, Tips, Contact) with smooth scrolling and active link highlighting.

Contact Form: A simple form for users to send messages, with basic validation and a thank-you alert.

Herbal Health Tips: A dedicated section with practical advice on using herbs safely.

Footer with Social Links: Placeholder social media icons for Facebook, Twitter, Instagram, YouTube, and LinkedIn.

**Technologies Used**

HTML5: Structure and content of the website.

CSS3: Custom styling, including responsive design, animations, and dark mode support. Utilizes Google Fonts (Inter) for typography.

JavaScript (ES6): Handles interactivity such as search filtering, read-more toggles, theme switching, form submission, and smooth scrolling.

Bootstrap 5: Framework for responsive grid system, components (e.g., navbar, cards, forms), and utilities.

Bootstrap Icons: Icon library for social media and theme toggle buttons.

Unsplash Images: High-quality, royalty-free images for the hero section and plant cards.
